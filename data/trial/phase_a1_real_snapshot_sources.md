## TWSE market data blocker

The 20-session TWSE market-data extract was removed because the previous version did not verify all official daily transaction values and incorrectly calculated `traded_value_twd` as closing price multiplied by volume. That is not an acceptable substitute for official TWSE daily transaction value.

Unresolved official TWSE items:

1. Official `2026-07-10` daily transaction value.
2. Official final `2026-07-15` close, traded share volume, and daily transaction value.
3. All 20 official TWSE daily transaction-value observations ending on `2026-07-15`.

Until those items are verified, this PR must not claim an `average_turnover_20d`, must not set `required_data_valid=true`, and must not run the real-data screening engine.

## Market-cap share-count blocker

Capital stock and par value imply issued shares:

- Capital stock: `259,323,700,670` TWD.
- Par value: `10` TWD.
- Issued shares: `259,323,700,670 / 10 = 25,932,370,067`.

However, this is not sufficient by itself for `common_shares_outstanding`. Treasury-share treatment and whether issued shares equal outstanding common shares as of `2026-07-15` remain unresolved. If the exact market-cap basis cannot be reconciled, the canonical row must remain blocked.
